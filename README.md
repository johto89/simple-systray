# simple-systray
a cross platfrom Go library to place an icon and menu in the notification area

## Requirements
- Go 1.9

## Usage
```
go get https://github.com/johto89/simple-systray
simple-systray
```

Manual:
```
git clone git@github.com:johto89/simple-systray.git
cd simple-systray
go get
go build
./simple-systray
```
