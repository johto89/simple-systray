# simple-systray
a cross platfrom Go library to place an icon and menu in the notification area

## Requirements
- Go 1.9

## Usage
```
go get github.com/Osuka42g/simple-systray
simple-systray
```

Manual:
```
git clone https://github.com/Osuka42g/simple-clock-systray.git
cd simple-systray
go get
go build
./simple-systray
```
